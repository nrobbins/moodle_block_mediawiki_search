NOTE: This block is intended to search stand-alone MediaWiki based wikis. If you just want a block to search Wikipedia, the Wikipedia block (in the Moodle Plugins Directory) is a much better choice.

==Installation==
1. Place the mediawiki_search folder in the blocks directory.
2. Visit the Notifications page and install the block.
==Configuration==
1. Turn editing on.
2. Add the block.
3. Click the Configuration icon in the heading of the block.
4. Type the title in the Title field.
5. Type the address of the MediaWiki installation in the Address field.
6. Type the path to the logo in the Logo field.
7. Click the Save Changes button.
